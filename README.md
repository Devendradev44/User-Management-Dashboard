# User Management Dashboard

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search
- Sort
- Pagination
- Filter Popup
- Responsive Design

## Tech Stack

- React
- TypeScript
- Axios
- Vite

## Setup

npm install

npm run dev

## Assumptions

JSONPlaceholder only provides 10 users.

POST/PUT/DELETE requests are mocked and do not persist after refresh.
